# **App Name**: Network Pilot

## Core Features:

- Real-Time Dashboard: Display server and tower status in real-time using cards with color-coded alerts for issues like weak signal or downtime.
- User Management: Add new PPPoE users by specifying username, password, server, speed, and expiry, automatically creating the account on the Mikrotik server.
- Automated Alerts: Continuously monitor tower signal strength and send immediate alerts via the dashboard when signal levels fall outside defined limits.

## Style Guidelines:

- Use a modern UI with a clean and strong design.
- Implement dark and light themes for user preference.
- Accent color: Teal (#008080) to provide a professional and modern feel.
- Employ a grid or flexbox layout for responsiveness across all screens.
- Utilize high-quality SVG icons instead of traditional images.
- Incorporate smooth transitions and animations for a professional feel.
